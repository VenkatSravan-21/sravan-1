🆘 COMMON ISSUES & SOLUTIONS

Issue: "npm not found"
Solution: Install Node.js from https://nodejs.org/

Issue: "git not found"
Solution: Install Git from https://git-scm.com/

Issue: "Changes not appearing on live site"
Solution: 
  1. Did you git push? (git push origin main)
  2. Check Render deployment status
  3. Hard refresh browser (Ctrl+Shift+Delete)

Issue: "Render deployment failing"
Solution:
  1. Check Render logs for errors
  2. Verify package.json is valid
  3. Ensure render.yaml is correct
  4. Check Node version compatibility

Issue: "Can't connect to GitHub from Render"
Solution:
  1. Verify repository is PUBLIC
  2. Generate GitHub personal access token
  3. Use token instead of password
